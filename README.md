# About project
This project is a term assignment for the Penetration Testing course. It was completed in a team of three. As part of the project, we conducted a simulated penetration test of Broken Crystals marketplace, which required the preparation of a Statement of Work, a threat model, and a final report.

For each of found vulnerabilities we prepared PDF document containing PoC organized according to STRIDE methodology. To complement the project, we prepared an example email notifying the client about critical vulnerabilities, based on the provisions outlined in the SoW.

The main objective of the project was to provide a comprehensive understanding of how penetration tests are conducted from start to finish, with particular emphasis on the ability to prepare  documentation.
